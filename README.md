# WireKit

WireKit is a UI component toolkit for Laravel Blade with a companion frontend layer. It provides a cohesive set of
building blocks for consistent interfaces and integrates with modern build tooling.

## What it includes

- Blade components for common UI patterns
- TypeScript/JavaScript behavior for interactive components
- Tailwind CSS-friendly styling conventions

## Getting started

### Prerequisites

- PHP (compatible with your Laravel version)
- Node.js and a package manager (npm, pnpm, or yarn)

### Install

From your Laravel app:

1. Add the package to your project.
2. Publish assets/config as needed.
3. Build frontend assets.

### Run

Use your normal Laravel and frontend dev workflows.
